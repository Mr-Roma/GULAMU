# Gulamu App

GulaMu adalah sebuah aplikasi yang berbasis mobile yang dapat mentrack pemasukan gula yang masuk kedalam tubuhmu.

## Installation

Clone project ini

```bash
https://github.com/Mr-Roma/Motionhack10.git
```

Mendapatkan dependency

```bash
flutter pub get
```

Menjalankan project

```bash
flutter run
```
